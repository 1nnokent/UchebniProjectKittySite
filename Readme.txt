Краткое описание проекта:
Проект представляет собой современную образовательную платформу, предназначенную для оптимизации учебного процесса. Система позволяет организовывать учебные группы, управлять заданиями и курсами, а также отслеживать успеваемость участников.

Инструкция по установке и развертыванию:
1. Клонируйте репозиторий:
   git clone https://github.com/your_username/educational_platform.git
2. Перейдите в каталог проекта:
   cd educational_platform
3. Установите необходимые зависимости:
   # Для установки всех библиотек, используемых в main.py, выполните следующую команду:
# Для установки всех зависимостей, используемых в main.py, выполните следующую команду:
# Flask — основной фреймворк для работы приложения.
# Flask-Login — библиотека для управления сессиями и аутентификацией.

pip install Flask Flask-Login
4. Настройте переменные окружения (создайте файл .env, скопировав настройки из .env.example).
5. Запустите приложение:
   python app.py
   // Альтернативно, можно запустить проект с помощью Docker:
   docker-compose up --build
